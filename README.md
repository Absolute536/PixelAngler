# Pixel Angler (Capstone Project 2)

## Description
This repository houses the source code flies and assets for the game Pixel Angler. 

It is a 2D, pixel-art style fishing game from a top-down perspective implemented using the Godot game engine. 

## Note For Examiners
For your assessment purposes:
* The source files are located in the repository itself.
* The latest exported project which can be run on your machine locally is located under the release section with the tag ```v0.0.1-prototype```.
  Just unzip the folder and run the (.exe).
* *P/S: Windows might give you a warning, because the application is not signed.*

###### *PP/S: Please ignore the nonsensical comments in the source code :sweat:.*
